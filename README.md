## Quickstart

```
  git clone https://github.com/LocoPixel/React-Redux-Crud-App
  cd your_App_
  npm install
  npm start

  To run backend
  cd your_App_
  cd Backend
  npm install
  npm start
```

**Note : Please make sure your MongoDB is running. at localhost:27017**
